# Minesweeper-Game-C-Console-Version-
This project is a fully functional Minesweeper game built using object-oriented programming in C++. It includes dynamic mine placement, adjacency calculations, and user interactions such as revealing cells and flagging mines. The game supports different difficulty levels, tracks elapsed time, and stores winners’ names and completion times in a file.

It demonstrates strong concepts in OOP, recursion, grid-based logic, file handling, and real-time user interaction in a console environment.
